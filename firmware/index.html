--- title: CANWork Firmware Updater description: Update your firmware with a Chrome Browser ---

This page is for updating the firmware on your CANWork board.

Directions and Notes:

  1. Make sure you are running an up-to-date version of Chrome, at least version 61 or newer.
  2. Hold down the Boot button.
  3. Plugin the board with the Boot button held.
  4. Release the Boot button.
  5. Click on the button below and select your target.

Your device has multiple DFU interfaces. Select one from the list below:

Runtime mode
DFU mode
Firmware Download (write to USB device)
